# Cocos2d-x Atals Unpacker

## Requirements

- Python (with pip)
- [Pillow](https://python-pillow.org)

## Installation

```sh
$ pip install -r requirements.txt
```
or
```sh
$ pip install Pillow
```

## Usage

`test.plist` and `test.png`:
```sh
$ ccplist2png.py test
```

or `*.plist` of same directory:
```sh
$ ccplist2png.py
```
